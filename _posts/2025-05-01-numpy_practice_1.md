### 6. np.mean(x, axis=(0,1)) 

```python
x = np.array([
    [[1, 2, 3], [2, 4, 8]],      # 첫 번째 그룹 (x[0])
    [[0, 0, 0], [1, 1, 1]]       # 두 번째 그룹 (x[1])
])

x.shape  # (2, 2, 3)


x[0][0] = [1, 2, 3]
x[0][1] = [2, 4, 8]
x[1][0] = [0, 0, 0]
x[1][1] = [1, 1, 1]

```

| 열 index | 요소       | 평균               |
| -------- | ---------- | ------------------ |
| 0        | 1, 2, 0, 1 | (1+2+0+1)/4 = 1.0  |
| 1        | 2, 4, 0, 1 | (2+4+0+1)/4 = 1.75 |
| 2        | 3, 8, 0, 1 | (3+8+0+1)/4 = 3.0  |

```python
np.mean(x, axis=(0,1))
# 결과: array([1.  , 1.75, 3.  ])
```

### 7. np.mean(x, axis=(1,2))

```python
x = np.array([
    [[1, 2, 3], [2, 4, 8]],      # 첫 번째 그룹 (x[0])
    [[0, 0, 0], [1, 1, 1]]       # 두 번째 그룹 (x[1])
])

x.shape  # (2, 2, 3)
```

axis=1: **행(2개)** 평균

axis=2: **열(3개)** 평균

 → 즉, 각 묶음(x[0], x[1]) 내부의 **전체 값 평균**을 계산

```python
# x[0] 
[[1, 2, 3],
 [2, 4, 8]]
 
모든 값: 1 + 2 + 3 + 2 + 4 + 8 = 20
평균: 20 / 6 = 3.333...

# x[1]
[[0, 0, 0],
 [1, 1, 1]]

모든 값: 0 + 0 + 0 + 1 + 1 + 1 = 3
평균: 3 / 6 = 0.5
```

```python
np.mean(x, axis=(1,2))  
# 결과: array([3.33333333, 0.5])
```

.

.

### 8. np.mean(x, axis=(0,2))

```python
x[0] =
[[1, 2, 3],
 [2, 4, 8]]
 
→ 열 평균 (axis=2):
[1, 2, 3] → 평균: (1+2+3)/3 = 2.0
[2, 4, 8] → 평균: (2+4+8)/3 = 4.666...

→ 결과: [2.0, 4.666...]



x[1] =
[[0, 0, 0],
 [1, 1, 1]]
 
→ 열 평균 (axis=2):
[0, 0, 0] → 평균: 0
[1, 1, 1] → 평균: 1

→ 결과: [0.0, 1.0]
```

### 

이제 axis=0 방향으로 평균

- (2.0 + 0.0) / 2 = 1.0
- (4.666... + 1.0) / 2 = 2.833...



```python
np.mean(x, axis=(0, 2))
# 결과: array([1.        , 2.83333333])
```
